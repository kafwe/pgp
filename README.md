# NIS Project
## Running
### Mac/Linux
Run `python3 main.py`

### Windows
Run `python main.py`

# Flags
## Reset Database and Keys
Run `python3 main.py --reset`

## Enable Verbose Logging for Debugging
Run `python3 main.py --debug`

# Tests
## Reset Database and Keys
Run `python3 test.py` or `python3 test.py --debug`
